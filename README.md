# Task Management Application

A modern task management application built with Vite, React, TypeScript, and shadcn/ui components. This application helps you organize and manage your tasks efficiently with a beautiful and responsive user interface.

## 🚀 Getting Started

### Prerequisites

Before you begin, ensure you have the following installed on your system:

- [Node.js](https://nodejs.org/) (v16 or later recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js) or [Yarn](https://yarnpkg.com/)
- [Git](https://git-scm.com/)

### 🛠 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/FAIZKHANMOHAMMED/Taskmanagement.git
   cd Taskmanagement
   ```

2. **Install dependencies**
   ```bash
   # Using npm
   npm install
   
   # Or using Yarn
   yarn install
   ```

### 🚦 Running the Application

#### Development Mode

To start the development server with hot-reload:

```bash
# Using npm
npm run dev

# Or using Yarn
yarn dev
```

This will start the development server at [http://localhost:5173](http://localhost:5173)

#### Building for Production

To create a production build:

```bash
# Using npm
npm run build

# Or using Yarn
yarn build
```

To preview the production build locally:

```bash
# Using npm
npm run preview

# Or using Yarn
yarn preview
```

## 🧪 Testing

To run linting:

```bash
# Using npm
npm run lint

# Or using Yarn
yarn lint
```
